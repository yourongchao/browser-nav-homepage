# 修改快捷方式样式

## 目标

* 移除所有 `<a>` 标签（nav-item）的背景底色

* 确保快捷方式仅包含圆框图标和位于图标下方的名称，无额外背景元素

* 保持视觉一致性

## 修改内容

### 1. 修改 `.nav-item` 样式（lines 221-237）

* 移除 `background`、`backdrop-filter`、`box-shadow` 属性

* 移除 `padding` 或大幅减少

* 移除 `border` 和 `border-radius`（除非需要保留）

* 调整 `transition` 效果

### 2. 保持 `.nav-item-icon` 样式

* 保留圆框样式（`border-radius: 50%`）

* 保留渐变背景

* 保持图标大小和居中对齐

### 3. 调整 `.nav-item-name` 样式

* 确保文字居中显示

* 保持适当的间距

### 4. 修改悬停效果（lines 291-295）

* 调整 `.nav-item:hover` 效果，移除背景相关变化

* 可保留缩放或边框效果

### 5. 检查响应式样式（lines 820-822, 846-848）

* 确保小屏幕上仍保持简洁样式

* 调整图标大小和文字大小

### 6. 检查暗色模式样式（lines 875-878）

* 移除暗色模式下的背景色

* 确保文字颜色合适

## 预期效果

* 快捷方式仅显示圆框图标和下方文字

* 无额外背景、阴影或边框

* 鼠标悬停时仅显示微妙效果

* 在所有屏幕尺寸和模式下保持一致

## 实现步骤

1. 编辑 `style.css` 文件
2. 修改 `.nav-item` 相关样式
3. 调整悬停和响应式效果
4. 验证修改后的显示效果
